# Resume Alignment: Product-level Advertising Growth Agent

## Why this project

该项目把本人美团实习中的商品级广告产品能力抽象成一个可运行的 AI Agent 系统。它展示的不只是业务理解，还包括 Tool Calling、RAG、SQLite 指标计算、可观测 Trace、自动化 Eval、消融实验和课程报告交付。

## Alignment with internship experience

| 实习经历关键词 | 项目对应模块 |
|---|---|
| POI 级广告升级商品级广告 | Product-level Ad Decision Agent |
| 主推品溢价 | Bid Strategy & Bid Multiplier Simulation |
| 智能调价 | ROI Guardrail Tool |
| PCVR、售价、历史 ROI | Bid Range Estimator |
| CVR + GMV 占比 | Product Growth Score |

| CTR、CVR、订单、ROI | Eval Metrics & Experiment Simulation |
